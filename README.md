# 3D Print Tracker
A flask app that simplifies the 3D printing process for lab techs at GW SEASCF.

## Installation 
#### Requirements
* Python requirements
*
* python 3
* flask
* rt
* python-dotenv

## Usage

run ``` gunicorn3 --workers=3 flask_app:app ```
